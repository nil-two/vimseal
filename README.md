# vimseal

8枚目のvimtutor。

https://unityroom.com/games/vimseal

## 操作方法

### 一般

| キー                   | 操作         |
|------------------------|--------------|
| 矢印キー, h, j, k, l   | 上下左右移動 |
| \<Enter\>, \<C-m\>     | 選択         |
| \<Backspace\>, \<C-h\> | 戻る         |
| \<Esc\>, \<C-[\>       | キャンセル   |

### ゲーム

| キー                 | 操作           |
|----------------------|----------------|
| h                    | 左に1文字移動  |
| j                    | 下に1文字移動  |
| k                    | 上に1文字移動  |
| l                    | 右に1文字移動  |
| 0                    | 行の先頭に移動 |
| $                    | 行の末尾に移動 |
| \<Enter\> or \<C-m\> | 次へ           |
